## 🚀 使い方

### 新規ユーザー向け

1. **環境診断:**
   ```bash
   bun run check:env
   ```

2. **自動セットアップ:**
   ```bash
   bun run setup
   ```

3. **GASで初期化:**
   - GASエディタで `initConfig()` を実行
   - GASエディタで `syncAllMetrics()` を実行

4. **スプレッドシートで確認:**
   - Dashboard シートでデータを確認

### 既存ユーザー向け

既存の設定は変更されません。新しいスクリプトは任意で使用できます。
